Research

Chip Design

This research was over the hardware for deep learning. The chip has a systolic array of processing elements that preformed the required calculations for more efficient deep learning. My part of that was placement, I created c++ code that took in a Verilog netlist file as well as a DEF file of the processing elements and made a def file with the most optimal placements for the DEF file. It required about parsing through files, graphs and communication with the other people working on other parts of the project.
